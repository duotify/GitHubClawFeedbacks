# GitHubClaw 龍蝦堡 — 問題回報區

歡迎來到 **GitHubClaw（龍蝦堡）** 的官方問題回報區！本 Repository 是同學們在學習過程中提交疑問、回饋與問題的專屬空間。

🔗 **課程連結**：[https://learn.duotify.com/courses/githubclaw](https://learn.duotify.com/courses/githubclaw)

## 🐞 如何提交問題或回饋

如果你在使用「龍蝦堡」的過程中遇到任何問題、發現錯誤，或是有任何建議，歡迎透過以下步驟提交 Issue：

1. 點擊上方的 **[Issues](../../issues)** 標籤
2. 點擊右上角的 **New Issue** 按鈕
3. 填寫問題標題與詳細描述，建議包含：
   - 📝 **問題的詳細描述**（如提示詞與意圖）
   - 🖼️ **截圖或錯誤訊息**（如適用）
   - 💻 **執行的偵錯記錄**（如適用）
4. 送出 Issue，課程團隊將盡快回覆

### ⚠️ 提交前請注意

- 請先**搜尋現有 Issues**，確認你的問題尚未被他人回報
- 請提供**足夠的細節**，以便課程團隊快速理解並協助你解決問題
- 本 Repository 僅供**課程相關問題**回報使用
