## 📖 課程簡介

**GitHubClaw（龍蝦堡）** 是由 [Duotify](https://learn.duotify.com) 推出的 GitHub 實戰課程。課程以生動活潑的方式，帶你從零開始掌握 GitHub 的核心功能與最佳實踐，讓你能夠在現代軟體開發團隊中游刃有餘地使用 GitHub 進行版本控制與協作開發。

---

## 🎯 課程學習目標

完成本課程後，你將能夠：

- ✅ 理解 **Git 版本控制**的核心概念與基本操作
- ✅ 熟練使用 **GitHub** 平台的各項功能
- ✅ 建立並管理 **Repository（儲存庫）**
- ✅ 使用 **Branch（分支）** 進行功能開發與版本管理
- ✅ 透過 **Pull Request（拉取請求）** 進行程式碼審查與合併
- ✅ 利用 **Issues** 進行專案管理與問題追蹤
- ✅ 了解 **GitHub Actions** 自動化 CI/CD 工作流程
- ✅ 掌握團隊協作的最佳實踐與工作流程

## 🐞 如何提交問題或回饋

如果你在課程學習過程中遇到任何問題、發現錯誤，或是有任何建議，歡迎透過以下步驟提交 Issue：

1. 點擊上方的 **[Issues](../../issues)** 標籤
2. 點擊右上角的 **New Issue** 按鈕
3. 填寫問題標題與詳細描述，建議包含：
   - 📌 **問題發生的章節 / 影片名稱**
   - 📝 **問題的詳細描述**
   - 🖼️ **截圖或錯誤訊息**（如適用）
   - 💻 **你的作業系統 / 環境資訊**（如適用）
4. 送出 Issue，課程團隊將盡快回覆

### ⚠️ 提交前請注意

- 請先**搜尋現有 Issues**，確認你的問題尚未被他人回報
- 請提供**足夠的細節**，以便課程團隊快速理解並協助你解決問題
- 本 Repository 僅供**課程相關問題**回報使用
